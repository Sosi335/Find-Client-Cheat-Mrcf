README

This project is a Python script that provides a list of Minecraft clients.

Requirements
- Python 3.11

Description
This script allows you to browse and find many different Minecraft clients.
There are over 200 clients included in the list.

Features
- Browse a large list of Minecraft clients
- Search bar to quickly find a specific client
- Scrollable interface

Notes
- The scrollbar is located on the right side of the interface.
- It may not be very visible depending on your system or theme, but it is present and works.

Usage
1. Make sure you have Python 3.11 installed.
2. Run the Python file.
3. Use the search bar to find a client or scroll through the list.
